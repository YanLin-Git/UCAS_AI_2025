# 三、DFS

- **不完备**：如果搜素树深度=$\infty$，会在某个分支，一直递归下去...
- **深度有限搜索**：最多往下搜索 $l$ 层
    - 解决了无限路径问题，但如果目标在 $d$ 层，$l<d$ 时，也找不到解

# 四、叠代深入深度优先搜索

1. 第一次限制深度为1，做一次**DFS**
2. 第二次限制深度为2，做一次**DFS**
3. 第二次限制深度为3，做一次**DFS**
4. ...
